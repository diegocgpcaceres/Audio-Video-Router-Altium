# Audio/Video Router PCB Design

Diseño completo de PCB para un sistema de distribución de audio y video, desarrollado como proyecto integrador de la materia **Diseño Asistido por Computadora (DAC)** de Ingeniería Electrónica.

El proyecto abarca la captura esquemática, diseño de PCB, generación de documentación de fabricación y modelado 3D utilizando Altium Designer.

## Tecnologías y Herramientas

* **Diseño PCB:** Altium Designer
* **Modelado 3D:** Herramientas integradas de Altium Designer
* **Documentación:** BOM (Bill of Materials), planos de fabricación y ensamblado

## Estructura del Repositorio

* `/Documentacion`: Contiene el informe técnico completo (PDF), incluyendo criterios de diseño, reglas utilizadas, documentación de fabricación y renders 3D.
* `/Proyecto_Altium`: Archivos fuente del proyecto desarrollados en Altium Designer.
* `/Imagenes`: Capturas del esquemático, PCB final y vistas 3D del diseño.

## Características del Proyecto

* PCB de dos capas.
* Más de 200 componentes electrónicos.
* Componentes SMD y Through-Hole.
* Generación automática de BOM.
* Modelado tridimensional del ensamblado final.
